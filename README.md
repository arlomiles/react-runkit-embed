# react-runkit-embed
Embed RunKit in your React apps

## How to use
`import RunKit from 'react-runkit-embed/RunKit'`

then

`<RunKit param=val...>your code here</RunKit>`

### Parameters
- readOnly
- nodeVersion
- env
- title
- minHeight
- packageResolutionTimestamp
- preamble
